# ESP32 BLE Automotive Telemetry Dashboard

## Overview
Real-time automotive telemetry dashboard developed using ESP32 BLE and web technologies.

This project simulates vehicle telemetry data and displays parameters such as speed, RPM, battery voltage, and temperature on a web-based dashboard.

## Features
- BLE-based wireless telemetry communication
- Real-time dashboard visualization
- Vehicle speed, RPM, voltage, and temperature monitoring
- Web interface using HTML, CSS, and JavaScript
- Automotive IoT concept demonstration

## Technology Stack
- ESP32
- Bluetooth Low Energy (BLE)
- HTML
- CSS
- JavaScript
- Embedded Systems
